# HSVColorTrackbar

HSV Color trackbar for finding an optimal range of lower and upper boundaries for certain color in OpenCV.

For Lower Boundary the Trackbars are [H,S,V]
For Upper Boundary the Trackbars are [H2,S2,V2]
